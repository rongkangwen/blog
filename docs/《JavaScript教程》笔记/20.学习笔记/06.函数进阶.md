# 函数进阶
## 闭包

__闭包__ 是指内部函数总是可以访问其所在的外部函数中声明的变量和参数，即使在其外部函数被返回（寿命终结）了之后

在javascript中，所有函数都是天生闭包（`new Function`创建的函数除外）

JavaScript 中的函数会自动通过隐藏的 `[[Environment]]` 属性记住创建它们的位置，所以它们都可以访问外部变量
## 箭头函数
:::note
+ 没有`this`
+ 没有`arguments`
+ 不能使用`new`进行操作
+ 它们也没有 `super`
:::
